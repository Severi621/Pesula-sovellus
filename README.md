# Pesula-sovellus

    Kalenteri näkymä 
    Mahdollisuus syöttää tilaus ja sen tiedot 
    Varaston tiedot 
    Automaattinen Pesuaika, kun tilaus palautetaan pesulalle 
    Mahdollisuus päivittää sovellusta tulevaisuudessa 
    Lisätä tuotteita varastoon ja tilauksiin 
    Tallentaa asiakkaiden tietoja 
    nopeuttaa tilauksien syöttelyä 
    Mahdollinen verkkokäyttöliittymä 
    Sovellusta voisi käyttää monella eri laitteella, siten ettei data olisi talletettu vain yhteen.
